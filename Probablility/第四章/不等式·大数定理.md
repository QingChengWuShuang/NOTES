# 第四章（1） 不等式，大数定理



## 一、不等式部分

### 1 马尔可夫不等式

$\boxed{马尔可夫不等式：设X是一个均值有限的非负随机变量，均值为E(X)，那么，对于任意正数a，有 \newline P(X \geq a) \leq \frac{E(X)}{a}，换言之，P(X<a) \geq 1- \frac{E(X)}{a}}$



### 2 切比雪夫不等式

$\boxed{切比雪夫不等式：设X是一个随机变量，均值E(X)，方差\sigma^2都是有限的。那么，对于任意的k>0，存在，P(|X-E(x)| \geq k\sigma) \leq \frac{1}{k^2}}$

**含义**：随机变量于均值超过k个标准差的概率，不会超过$\frac{1}{k^{2}}$

## 三、大数定律

### 1 切比雪夫大数定律

内容：$随机变量X_1,X_2，...,X_n两两不相关，算数平均值\bar{X_n}依概率收敛于期望的均值$

公式：$\boxed{若随机变量X_1,X_2,...,X_n两两不相关，方差D(X_i)存在且有限， \lim_{n \rightarrow \infty }{P(|\frac{1}{n}\Sigma_{i=1}^{n}{X_{i}} - \frac{1}{n}\Sigma_{i=1}^{n}E(X_{i})| < \epsilon) } = 1  ，对于任意的\epsilon > 0 均成立。}$

记作：

$\frac{1}{n}\Sigma_{i=1}^{n}{(X_i-E(X_i)) \stackrel{P}{\longrightarrow} 0}$



### 2 伯努利大数定律

公式： 设$n_A$为n重伯努利实验中事件A发生的次数，p为A发生的概率

$\boxed{\lim_{n \rightarrow \infty}{P(|\frac{n_A}{n}-p| < \epsilon} = 1} $





### 3 辛钦大数定律

公式：随机变量序列$\{ X_{n} \}$独立同分布，且$E(X_i) = \mu$存在，对于任意的$\epsilon > 0$，有

$\boxed{\lim_{n \rightarrow \infty}P(|\frac{1}{n}\Sigma_{i=1}^{n}X_i - \mu | < \epsilon) = 1}$
